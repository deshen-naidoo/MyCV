# Online Professional CV

A responsive, professional online curriculum vitae designed to highlight career experience, education, and core competencies.

## Project Overview
This repository contains the source code for my personal web portfolio, developed as part of Practical Task 2. It strikes a balance between professional formatting and personal branding, acting as a live digital footprint of my resume.

## Live Website
You can view the live deployed CV here: 
[https://deshen-naidoo.github.io/MyCV/](https://deshen-naidoo.github.io/MyCV/)

## Features & Technologies
* **HTML5:** Semantic structuring of professional experience and education.
* **CSS3 & Bootstrap 5:** Utilized for a clean, responsive layout that adapts to both desktop and mobile viewing.
* **GitHub Pages:** Used for continuous deployment and static hosting.
